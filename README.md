# 廖添 - 个人主页

一个简约高级的个人网站，展示个人信息、荣誉成就和运动数据。

## 特点

- 🎨 淡蓝色主题设计
- 📱 完全响应式布局
- ✨ 流畅的动画效果
- 🚀 纯HTML/CSS/JavaScript，无需构建工具

## 如何使用

1. 直接双击 `index.html` 文件在浏览器中打开
2. 或者使用本地服务器运行（推荐）

## 文件结构

```
liaotian-portfolio/
├── index.html      # 主页面
├── style.css       # 样式文件
├── script.js       # 交互脚本
└── README.md       # 说明文档
```

## 网站板块

- **首页** - 个人简介和亮点数据
- **关于我** - 基本信息、在校职务、兴趣爱好
- **荣誉成就** - 时间线展示所有获奖记录
- **运动数据** - 各项运动能力数据展示
- **联系方式** - 邮箱和社交媒体

## 技术栈

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript
- Font Awesome Icons

## 浏览器支持

- Chrome (推荐)
- Firefox
- Safari
- Edge
